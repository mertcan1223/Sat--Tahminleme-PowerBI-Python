Satış Tahminleme ve İş Zekası Projesi
Bu proje, bir işletmenin geçmiş satış verilerini analiz ederek gelecek dönem taleplerini %85+ doğrulukla tahmin etmek amacıyla geliştirilmiştir.

🛠 Kullanılan Teknolojiler
Dil: Python (Pandas, Numpy)

Makine Öğrenmesi: Scikit-learn (Random Forest Regressor)

Görselleştirme: Power BI

📈 Proje Aşamaları
Veri Hazırlama: Eksik veriler giderildi ve trend analizi yapıldı.

Özellik Mühendisliği: Modelin başarısını artırmak için Lag Features (Gecikmeli Veriler) ve Rolling Mean (Hareketli Ortalama) kullanıldı.

Model Eğitimi: Random Forest algoritması ile hiperparametre optimizasyonu yapıldı.

Raporlama: Tahmin sonuçları Power BI'a aktarılarak interaktif bir dashboard oluşturuldu.
